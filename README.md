# 📋 CheatSheets

Eine kuratierte Sammlung von Spickzetteln zu den wichtigsten Programmiersprachen und Technologien — ideal als schnelle Referenz im Entwickleralltag.

## Inhaltsverzeichnis

| # | Thema | Beschreibung |
|---|-------|--------------|
| 1 | [RegEx](RegEx.md) | Reguläre Ausdrücke — Syntax, Quantifier, Gruppen, Lookahead/Lookbehind |
| 2 | [C#](CSharp.md) | C# Sprachreferenz — umfassend mit Beispielen |
| 3 | [Flutter](Flutter.md) | Flutter/Dart Grundlagen und Widget-Übersicht |
| 4 | [TypeScript](TypeScript.md) | TypeScript Typsystem, Generics, Utility Types |
| 5 | [JavaScript](JavaScript.md) | JavaScript Kernkonzepte und moderne Syntax |
| 6 | [SQL](SQL-Syntax-QRMs-DesignPatterns.md) | SQL-Syntax, ORMs und Design Patterns |

## Verwendung

Jeder Cheat Sheet ist eine eigenständige Markdown-Datei, die direkt auf GitHub gelesen oder lokal geklont werden kann:

```bash
git clone https://github.com/Sebastian90Sonntag/CheatCheets.git
```
